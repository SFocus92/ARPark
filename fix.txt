Абсолютно правильно! Тап — это только fallback для тестов. Основная логика — **реальное распознавание NFT-маркеров через камеру**. Давай добьём, чтобы MindAR работал без тапов.

## Текущая проблема

MindAR не загружается с CDN (404 на UMD). Нужно:
1. Скачать MindAR локально
2. Правильно инициализировать с твоими `.mind` файлами

## Пошаговое решение

### Шаг 1: Скачай MindAR локально

Выполни в терминале **прямо сейчас**:

```powershell
cd D:\ar-quest-sevapark

# Создай папку
New-Item -ItemType Directory -Force -Path public\js

# Скачай MindAR ESM
Invoke-WebRequest -Uri "https://cdn.jsdelivr.net/npm/mind-ar@1.2.5/dist/mindar-image-three.prod.js" -OutFile "public\js\mindar-image-three.prod.js"
```

Или просто открой в браузере https://cdn.jsdelivr.net/npm/mind-ar@1.2.5/dist/mindar-image-three.prod.js и сохрани как `public/js/mindar-image-three.prod.js`.

### Шаг 2: Замени `ar-scene.tsx` полностью

Вот продакшн-версия **без тапов, только реальное распознавание**:

```tsx
'use client';

import { useEffect, useRef, useState } from 'react';
import { useQuest } from '@/hooks/use-quest';
import { STEPS, QuestStep } from '@/lib/quest-config';

interface ARSceneProps {
  onReady?: () => void;
  onError?: (error: string) => void;
}

export function ARScene({ onReady, onError }: ARSceneProps) {
  const containerRef = useRef<HTMLDivElement>(null);
  const [isLoading, setIsLoading] = useState(true);
  const [status, setStatus] = useState('Инициализация...');
  const [error, setError] = useState<string | null>(null);
  const initRef = useRef(false);
  const scannerRef = useRef<any>(null);

  const {
    handleMarkerFound,
    setCameraReady,
    setCameraError,
    currentMarker,
    showingContent,
    completedSteps,
  } = useQuest();

  useEffect(() => {
    if (initRef.current) return;
    initRef.current = true;

    async function init() {
      try {
        // 1. Ждём A-Frame (он грузится в page.tsx)
        setStatus('Ожидание A-Frame...');
        await waitForGlobal('AFRAME', 10000);
        console.log('[AR] A-Frame готов');

        // 2. Ждём THREE.js
        setStatus('Ожидание THREE.js...');
        await waitForGlobal('THREE', 5000);
        console.log('[AR] THREE готов');

        // 3. Загружаем MindAR локально
        setStatus('Загрузка MindAR...');
        const mindarModule = await import('/js/mindar-image-three.prod.js');
        console.log('[AR] MindAR модуль загружен');

        // MindAR экспортирует классы. Ищем MindARImage
        const MindARImage = 
          (mindarModule as any).default?.MindARImage ||
          (mindarModule as any).MindARImage ||
          mindarModule;

        if (!MindARImage) {
          throw new Error('MindARImage не найден в модуле');
        }

        // 4. Готовим список маркеров
        const targets = STEPS
          .filter(s => s.markerType === 'nft' && s.nftDescriptor)
          .map(s => s.nftDescriptor!);

        console.log('[AR] Маркеры для загрузки:', targets);

        // 5. Создаём сканер
        setStatus('Создание сканера...');
        const scanner = new MindARImage({
          imageTargetSrc: targets.join(','),
          filterMinCF: 0.1,
          filterBeta: 0.001,
        });

        // 6. Обработчик обнаружения маркера
        scanner.onTargetFound = (targetIndex: number) => {
          console.log(`[AR] 🎯 Маркер найден! Индекс: ${targetIndex}`);
          
          // Находим соответствующий шаг
          const step = STEPS.find((s, i) => i === targetIndex);
          if (step) {
            console.log(`[AR] Найден: ${step.title}`);
            handleMarkerFound(step.id);
            setStatus(`✅ ${step.title}`);
          }
        };

        scanner.onTargetLost = () => {
          setStatus('🔍 Ищите маркер...');
        };

        // 7. Запускаем сканер в контейнере
        setStatus('Запуск камеры...');
        await scanner.start(containerRef.current!);

        scannerRef.current = scanner;
        console.log('[AR] ✅ Сканер запущен!');

        setIsLoading(false);
        setCameraReady(true);
        onReady?.();
        setStatus('📷 Наведите камеру на маркер');

      } catch (err: any) {
        console.error('[AR] ❌ Ошибка инициализации:', err);
        setError(err.message || 'Неизвестная ошибка');
        setCameraError(err.message);
        onError?.(err.message);
        setIsLoading(false);
      }
    }

    init();

    return () => {
      // Очистка при размонтировании
      if (scannerRef.current?.stop) {
        scannerRef.current.stop();
      }
    };
  }, []);

  // Текущий шаг для отображения прогресса
  const currentStep = STEPS[completedSteps];
  const displayStep = currentMarker && showingContent
    ? STEPS.find(s => s.id === currentMarker)
    : null;

  return (
    <div
      ref={containerRef}
      style={{
        position: 'fixed',
        top: 0,
        left: 0,
        width: '100%',
        height: '100%',
        zIndex: 0,
        backgroundColor: '#000',
        overflow: 'hidden',
      }}
    >
      {/* Status Bar */}
      <StatusBar status={status} />

      {/* Progress */}
      <ProgressBar 
        current={completedSteps} 
        total={STEPS.length} 
        title={currentStep?.title} 
      />

      {/* 3D Content when marker found */}
      {displayStep && <MarkerOverlay step={displayStep} />}

      {/* Loading Screen */}
      {isLoading && <LoadingScreen status={status} />}

      {/* Error Screen */}
      {error && <ErrorScreen error={error} />}

      <style jsx global>{`
        @keyframes spin {
          from { transform: rotate(0deg); }
          to { transform: rotate(360deg); }
        }
        @keyframes fadeIn {
          from { opacity: 0; transform: translateY(20px); }
          to { opacity: 1; transform: translateY(0); }
        }
      `}</style>
    </div>
  );
}

function waitForGlobal(name: string, timeoutMs: number): Promise<void> {
  return new Promise((resolve, reject) => {
    const start = Date.now();
    const check = setInterval(() => {
      if ((window as any)[name]) {
        clearInterval(check);
        resolve();
      } else if (Date.now() - start > timeoutMs) {
        clearInterval(check);
        reject(new Error(`Timeout waiting for ${name}`));
      }
    }, 100);
  });
}
```

### Шаг 3: Удали все тапы из `page.tsx`

Найди `handleStart` в `src/app/page.tsx` и упрости:

```typescript
const handleStart = useCallback(() => {
  // Проверка HTTPS
  if (typeof window !== 'undefined') {
    const protocol = window.location.protocol;
    const hostname = window.location.hostname;
    if (protocol !== 'https:' && hostname !== 'localhost') {
      setCameraError('Требуется HTTPS');
      return;
    }
  }

  startQuest();
  setShowAR(true);
}, [startQuest, setCameraError]);
```

## Как будет работать:

1. Пользователь нажимает "Начать квест"
2. Загружается A-Frame → THREE.js → MindAR (локально)
3. MindAR загружает 7 твоих `.mind` файлов из `public/assets/nft/`
4. Открывается камера
5. Пользователь наводит на реальный объект парка (который ты фоткал)
6. MindAR сравнивает картинку с камеры с `.mind` дескрипторами
7. При совпадении → `onTargetFound` → показывает 3D-объект
8. **Никаких тапов! Только реальное распознавание!**

После деплоя проверь консоль — должно быть:
```
[AR] Маркеры для загрузки: ["/assets/nft/marker-1", "/assets/nft/marker-2", ...]
[AR] ✅ Сканер запущен!
```

И при наведении на маркер:
```
[AR] 🎯 Маркер найден! Индекс: 0
[AR] Найден: Главные ворота
```
