🎯 СЛЕДУЮЩИЕ ШАГИ:

1. Запустить dev сервер: npm run dev
2. Открыть https://localhost:3000 (нужен HTTPS для камеры)
3. Протестировать AR-сканирование с маркерами из public/assets/nft/
4. При необходимости настроить quest-config.ts

⚠️ ВАЖНО:

- Для работы AR нужен HTTPS (кроме localhost)
- Маркеры должны быть в формате .mind (не .fset/.iset)
- Используйте Safari на iOS для доступа к камере
- Все тексты на русском языке
